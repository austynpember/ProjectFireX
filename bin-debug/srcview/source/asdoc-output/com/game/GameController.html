init()method 
public function init(e:Event = null):void

Main initialization function of the gameController. This starts by initializing all of the Game Objects like the player, levels, and the scene itself. Next is the 3D camera, then is the level initializer. Finally, set up all of the listeners for user input, damage, and other events.
