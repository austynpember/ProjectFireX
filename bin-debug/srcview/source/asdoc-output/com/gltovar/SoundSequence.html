This object handles playing sound with defined delays in the defined sequence. This class relys on the soundmanager from http://evolve.reintroducing.com/2008/07/15/as3/as3-soundmanager/ Make sure to read up on that site to see how to initialize sound in Matt Przybylski's code above.
